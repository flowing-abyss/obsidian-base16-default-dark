# Base16 Default Dark

A clean Obsidian theme based on the [Base16 Default Dark](https://github.com/chriskempson/base16) color scheme.

![Theme Preview](https://github.com/flowing-abyss/obsidian-base16-default-dark/blob/main/assets/theme-preview.png)

## Features

- Classic dark background with Base16 palette
- Orange headings, blue internal links, purple external links
- Styled callouts

## Installation

**Obsidian:** Settings → Appearance → Themes → Manage → Search "Base16 Default Dark"

**Manual:** Download [latest release](https://github.com/flowing-abyss/obsidian-base16-default-dark/releases) → Place in `.obsidian/themes/Base16 Default Dark/`
